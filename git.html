Git Kommandon


        
Här är några användbar git kommandon man använder i terminalen:


        

            
                

                    Kommando
                    Vad det gör
                

            
            

                
                    
git init .

                		
                 skapar ett tomt git repository
            

            

                
                    
git status

                		
                 ser status för ditt repository
            

            

                
                    
git add [filnamn]

                		
                 lägger till en fil för att uppdatera ditt repository
            

            

                
                    
git commit -m [meddelande]

                		
                 sparar ändringar till repository med ett meddelande
            

            

                
                    
git log

                		
                
                     ser commit historik

                     (använd --oneline för att se en kortare log)
                		
            

            

                
                    
git remote add origin main [github url]

                		
                 länkar ditt lokala repo till ditt nyskapade GitHub repo
            

            

                
                    
git push

                		
                
                     skickar ändringar till ditt git repository

                     används vanligtvis med -u origin main

                     -f / --force tvingar ett push kommando
                		
            

            

                
                    
git fetch

                		
                 hämtar från ett repository
            

            

                
                    
git pull

                		
                 uppdaterar din local branch från en remote branch
            

            

                
                    
git clone [repo url]

                		
                 klonar ett befintlig repo till din local branch
            

            

                
                    
git diff

                		
                 visar skillnader mellan dina branches
            

            

                
                    
git reset [commit id]

                		
                 återställ till en tidigare commit
            

            

                
                    
git restore [filnamn]

                		
                 återställ filen med en tidigare commit
            

            

                
                    
git branch

                		
                
                     visar existerande branches

                     använd git branch [ny branch namn] för att skapa en ny branch
                		
            

            

                
                    
git checkout [branch namn]

                		
                 byter till en annan branch
